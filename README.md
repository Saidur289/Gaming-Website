# Chill Gamer Project

## Project Overview
**Chill Gamer Project** is a web application where users can review games and get information about various games easily. The platform is designed to provide an easy way to explore games across all categories and share your reviews securely.

## Key Features
- **User Authentication:** Users must sign in to leave a review. Reviews are private and visible only to the user who created them.
- **Game Reviews:** Users can submit reviews for any game using a simple form.
- **Secure User Data:** All user data and reviews are stored securely in MongoDB.

## Technologies Used
- **Frontend:** React.js
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Authentication:** Firebase Authentication
